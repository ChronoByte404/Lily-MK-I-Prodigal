# Lily-MK-I-Prodigal
The first completed version of my Lily Framework.

## How to install?

In your *MacOS or Linux* terminal, run:

gh repo clone Sanware/Lily-MK-I-Prodigal

cd /Lily-MK-I-Prodigal

python3 -m pip install -r requirements.txt
